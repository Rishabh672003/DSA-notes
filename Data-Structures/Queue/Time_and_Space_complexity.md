## Time and Space Complexity:

| Operation   | Time Complexity | Space Complexity |
|-------------|----------------|------------------|
| enqueue     | O(1)           | O(1)             |
| dequeue     | O(1)           | O(1)             |
| front       | O(1)           | O(1)             |
| isEmpty     | O(1)           | O(1)             |
| size        | O(1)           | O(1)             |
